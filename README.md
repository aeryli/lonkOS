# lonkOS
a semi-fork of linkOS, with a Debian-style UI instead of an Ubuntu-style UI.

## building
Run the following commands.

<details><summary>dependencies</summary>
    > <details><summary>Windows</summary>
    > ```
    >     winget install nodejs
    >     node -v
    >     npm install linkedom
    >     npm install ws
    > ```
    > </details><details><summary>Linux</summary>
    > ```
    >     sudo apt install nodejs
    >     node -v
    >     npm install linkedom
    >     npm install ws
    > ```
    > </details>
</details><details><summary>build</summary>
    ```
        git clone https://github.com/aeryli/lonkOS.git
        cd lonkOS
        npm run build
    ```
</details><details><summary>run</summary>
    ```
        cd builds
    ```
</details>
